WEBVTT Nessa etapa, você vai aprender a fazer requisições HTTP para uma API e consumir então os dados dessa API utilizando o Fetch, que é a estratégia padrão para consumir API no JavaScript e fazer requisições HTTP. Além de ler dados vindo de uma API, você também vai aprender a enviar dados para essa API, por exemplo, para cadastrar um novo produto e assim por diante. Então, vamos lá, porque a partir de agora, você vai aprender a utilizar o Fetch.
