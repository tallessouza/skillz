WEBVTT Nessa aula eu vou mostrar para você como fazer a acertação de tipo. A acertação de tipo é um recurso muito interessante do TypeScript para você utilizar quando o TypeScript não sabe qual é a tipagem de um determinado objeto e aí você pode dizer assim para o TypeScript, confia, vem aqui que a tipagem é essa. Então é muito utilizado, por exemplo, para consumir API. Então digamos que você foi lá, consumiu uma rota da API, buscou pelas informações do perfil do usuário e aí quando essas informações chegam, quando a API devolve essas informações através da resposta, o TypeScript pode não conhecer aquele tipo e aí você pode usar a acertação para dizer qual é o tipo como uma espécie de conversão. Então para te mostrar esse exemplo eu vou criar primeiro uma variável que eu vou chamar ela de UserResponse e digamos que esse UserResponse seja esse objeto aqui, o TypeScript não sabe qual é o conteúdo dele, tanto que eu posso colocar aqui um UserResponse e ponto, tá vendo que não me aparece nada, não me aparece nenhuma opção aqui, porque o TypeScript não conhece quais são as opções disponíveis dentro desse UserResponse, eu deixei o objeto aqui vazio de propósito. Então eu posso fazer uma sessão desse tipo e dizer para o TypeScript o que é que tem aqui dentro. Então eu posso criar um type, eu vou chamar de UserResponse, lembre sempre de criar tipos com letra maiúscula, igual, abro aqui chaves e vou definir aqui a tipagem, então vai ter um id, por exemplo, que vai ser um número e vou colocar um name que vai ser aqui uma string, tá. Vou colocar um avatar também, que vai ser uma string e show, beleza. Então agora, olha o que eu posso fazer, para fazer uma sessão eu coloco um as e UserResponse, que é o tipo. Então quando eu quero fazer uma sessão de tipo, eu coloco as e o tipo, ou seja, eu estou dizendo assim, esse objeto, ele deve ser conforme o UserResponse, ou o padrão disso daqui é um UserResponse. E agora, se eu pegar o meu UserResponse e colocar um ponto, olha lá, me aparecem as propriedades, avatar, name e id, na verdade, eu coloquei o id, pronto, agora sim, avatar, id e name. Então isso daqui que é a sessão de tipo, é como se fosse uma conversão, então é como se eu estivesse convertendo o tipo desse objeto, que não tem nenhum tipo, para este daqui. Então esse recurso, ele é muito utilizado para consumir API, por exemplo, ou quando você quer fazer uma espécie de conversão, ou dizer para o Typescript qual é o tipo, então você está fazendo uma sessão, você está dizendo assim, Typescript, confia que o tipo desse cara aqui é isso daqui, esse é o padrão deste objeto. Então nessa aula você aprendeu a fazer a sessão de tipo.
