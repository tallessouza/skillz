WEBVTT Vamos falar sobre aplicativos web ou web application. Web application significa um aplicativo web, um programa na web. Então é um programa, um software que ele vai viver lá no servidor. Então você vai precisar de linguagens de programação, você vai precisar de bancos de dados. Ele é utilizado através das páginas web e pelo navegador. Então quando você abrir o navegador, você vai encontrar um aplicativo web provavelmente dependendo de onde você está indo. Quer ver um exemplo? Você está assistindo essa aula pelo aplicativo web da Rocketseat. Então se você olhar a URL, ele é app.rocketseat.com.br. Então é um aplicativo web, é uma página, você vê ele como uma página, você vê ele no navegador, só que ele é complexo e possui mais requisitos do que uma página simples ou do que um site. Por exemplo, um Facebook, um Gmail, um YouTube, um Figma, todos eles são aplicativos web, inclusive todo esse sistema que você usa para estudar aqui na Rocketseat, eles são bem complexos. Eles envolvem linguagem de programação, ele envolve banco de dados, envolve algo bem complexo e ele vive dentro de um servidor e você pode acessar ele de maneiras que você já entendeu até agora. Percebe? Então é legal você entender as diferenças de um site, um website, uma webpage, um aplicativo web, que é uma coisa bem mais robusta. Legal?
