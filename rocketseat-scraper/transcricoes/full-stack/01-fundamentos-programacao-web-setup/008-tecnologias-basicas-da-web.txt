WEBVTT Nessa aula vamos falar sobre tecnologias básicas da web, que são linguagens e ferramentas para você programar na web. Quando a gente fala ali do front-end, a gente tem três principais fundamentais, pode encontrar outras coisas dentro desse universo, mas essas três aqui você precisa saber. Que é o HTML, é uma linguagem de marcação de texto, para você estruturar o texto, criar links, colocar imagem, vídeo. A grande sacada do HTML é que você, ao invés de ter só texto puro, como se você tivesse na página de um livro, você pode colocar ali, por exemplo, um link que você clica e ele vai para outra página. Só que o HTML por si só é feio, apesar dele estruturar todo o seu texto, ele é feio. Então você vem com uma linguagem de estilo para deixar esse HTML bonito, que é o CSS, Cascading Style Sheet. No momento oportuno você vai se aprofundar, mas aqui é bom você entender que essa tecnologia, essas duas juntas aqui, é o que faz a beleza de quando você entra num site e ele tá bonito ou ele tá feio, justamente por causa dessas duas coisas. Agora, quando você precisa interagir no site, você tem uma linguagem de programação super usual no navegador. Ele já faz parte do seu navegador, o JavaScript está dentro do seu navegador. E tudo que você usar dessa linguagem de programação, que você vai também aprender em momento oportuno, você vai ver que ele vai servir como uma inteligência para a estrutura e para a beleza. Faltou então a inteligência, tem o JavaScript para isso. E aqui no back-end, você pode escolher diversas linguagens de programação para você executar lá no back-end. Você vai ter muitas coisas. O Node.js, por exemplo, é para você rodar o JavaScript dentro de um computador. Então o JavaScript, por padrão, ele está no browser, ele está no navegador. Está ali no seu Chrome, Firefox, Safari, etc. Para você rodar sem precisar daqueles carinhas, você instala um Node.js na sua máquina. Você pode instalar a BAN, que é uma outra. E aí você vai rodar o JavaScript no back-end e conseguir fazer todas as estruturas de back-end necessárias. Porém, linguagens para back-end tem muito mais. Java, Python, C Sharp, tem muitas linguagens para back-end. Uma das outras linguagens que você vai precisar é uma linguagem de consulta para banco de dados. Aqui você guarda dados em um banco de dados, é uma linguagem super segura. E aí você, além de guardar os dados ali, você consegue consultá-los, pegar esses dados de volta. Então, uma maneira segura de você guardar dados lá no seu back-end, através do seu programa e tudo mais, é o SQL. Você tem, além do SQL, você tem o NoSQL também, que é N-O-S-Q-L. Tudo em momento oportuno você vai estudar. Mas são maneiras bem bacanas de você guardar dados e proteger dados dentro do back-end.
