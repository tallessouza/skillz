WEBVTT Vamos falar sobre a diferença de WebPage e WebSite, são duas palavras que você vai encontrar bastante nesse universo da programação. WebPage é uma página, page, na web. Essa página é composta de alguns arquivos como HTML, CSS, Javascript, e você acessa essa página pelo seu navegador de preferência, Chrome, Firefox, Safari, etc., colocando a URL. Quer um exemplo? Eu vou lá, coloco essa URL, e aí eu acesso pelo meu navegador, e ele vai me devolver os arquivos de HTML, CSS, Javascript, etc., ok? Essa é uma página web. Entretanto, um WebSite é o agrupamento estruturado de páginas na web. Então ao eu entrar num site mais completo, como o rocketcity.com.br, eu vou encontrar ali a página inicial, chamada de home, casa, é a página inicial, e ali eu posso encontrar informações, por exemplo, sobre, eu poderia ir para um catálogo que vai mostrar alguns produtos, eu poderia ir para um blog, cada um disso daqui é uma WebPage, uma WebPage que está levando para outra WebPage, uma que vai levar para outra, que vai levar para outra, etc. Então, esse agrupamento de páginas da web é justamente chamada de WebSite, ou sitio, traduzindo, é um sítio, sítio, como se fosse uma fazenda, etc., eu não sei porque que inventaram dessa forma, mas é bem engraçado, né? Então sítio na web. Dessa forma, então, você tem um agrupamento de páginas, você tem um site, um WebSite, bacana?
