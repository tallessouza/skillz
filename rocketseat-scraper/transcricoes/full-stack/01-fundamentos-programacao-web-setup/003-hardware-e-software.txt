WEBVTT Vamos falar sobre hardware e software e vamos ver se você conhece o seu computador. Gente, quando a gente fala de hardware, é tudo que você pode pegar. Imagina comigo, você pode pegar o seu teclado, o seu mouse, sua câmera, as placas do seu computador, as memórias que tem lá dentro, HD. Bom, algumas pessoas vão dizer, não, eu não posso pegar porque está fechado. Tá certo, mas você pode tocar, né? Se você pode tocar, pode pegar. Provavelmente é um hardware, ok? Então é a parte física das coisas. Tem toda uma engenharia para isso, com certeza pessoas que estudam para construir esses componentes e tudo mais. Bacana? Agora, tem o software, que é a parte que controla o hardware. Então é a parte lógica do seu computador. E software, pessoal, geralmente é o sistema controlando o sistema. Coisas que a gente não vai estar ali pegando, fazendo, usando o software. Mas, aqui no Brasil, aqui no português, a gente às vezes chama de programa, software também. Lá nos Estados Unidos, do inglês, program e software são coisas diferentes. Porque é um programa, um aplicativo, um app, que é muito importante você também saber esses nomes. Geralmente é aquilo que o usuário vai conseguir interagir com. Então se eu consigo interagir aqui com esse programa, que eu escrevo as coisas, eu estou interagindo com esse aplicativo, com esse app, com esse programa. Se eu não interajo, por exemplo, quando eu clico no botão para fazer a impressão de uma coisa no computador. Ou quando eu clico para ligar o meu computador. Bom, de fato eu cliquei ali no hardware, cliquei naquele botãozinho para ligar o meu computador. Ou apertei aqui no meu sistema alguma coisa para fazer a impressão de alguma coisa. Só que está acontecendo coisas por trás, as quais o sistema está conversando com o sistema. E aí é o software. É só para você entender esses dois níveis de diferença. Mas não precisa memorizar, não precisa se preocupar com isso, fica bem tranquilo. O que você precisa mais entender aqui é sobre programas, aplicativos e apps, porque você vai construir essa parte aqui. Você provavelmente não será capaz de construir softwares com tudo o que você vai aprender aqui na programação web. A ideia da programação web não é construir softwares, mas construir programas e aplicativos, aquilo que a pessoa vai tocar. Por exemplo, quando você abre o seu navegador, quando você abre Chrome, Firefox, Edge, Safari no seu celular, qualquer coisa assim, Google para pesquisar, você está abrindo um programa chamado navegador. E ali você está usando esse sistema para navegar na internet. Você pode usar programas de mensagem, como o WhatsApp, por exemplo. Você pode usar redes sociais, como o Instagram. Então tudo isso são aplicativos, são apps. E aí a partir deles, você pode interagir o tempo inteiro com eles. É isso aí que você vai ser capaz de construir tudo pela internet. Então nessa aula você entendeu bem. O software vai ser necessário para controlar o hardware. Então para eu entender todas as teclas que eu estou clicando no meu teclado, para o mouse entender esse movimento aqui, existe um software por trás disso controlando ele. Fazendo a conexão entre essas duas coisas. E aí você entendeu também que o hardware é tudo que você pode tocar, pode pegar. Até aquilo que você não pode porque está fechado no computador. Mas é aquela parte física das coisas, enquanto o software é a parte não física. Também é uma maneira de você fazer programação. Mas aqui dentro dessa nossa área, a gente vai olhar para isso aqui.
