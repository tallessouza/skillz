WEBVTT Nessa aula vamos entender o conceito de array e aí daqui a pouco a gente vai colocar em prática. Primeiro vamos para a definição. Um array é uma coleção ordenada de valores, porque você tem uma coisa depois da outra e a gente pode até comparar o array com uma lista, uma lista onde cada item tem uma posição específica e aí essa posição específica quando a gente fala de array ela é conhecida como índice, certo? Então essa é a definição de array. Vamos dar uma olhada em um exemplo aqui para fazer uma ilustração. Pensa em uma caixa de correio que tem vários compartimentos numerados. Vamos imaginar que cada número significa que é um apartamento, por exemplo. Então você tem lá 100 apartamentos, então cada apartamento tem a sua caixa, onde vai receber ali as correspondências, as cartas e tudo mais. E aí dentro de cada compartimento a gente pode armazenar alguma coisa, aqui para o nosso exemplo o que a gente vai armazenar é uma carta. E aí pra pessoa saber qual é o compartimento que ela deve acessar, aí que entra os índices, porque os compartimentos como eles são numerados de forma ordenada, começando do zero e não do 1, isso daqui também é uma outra característica do array, sempre começa do zero e não do 1 e vai até um determinado número. E aí esse número final, então imagina se eu tenho 100 compartimentos de carta, 100 caixas para colocar carta, então eu tenho do índice 0 até o 99, por isso que a gente sempre fala que o total de um array, o total de compartimentos é menos 1, porque como ele começa do zero, então se eu tenho 100 compartimentos e esses compartimentos é um array e começa do índice 0, então vai até o 99, são 100 posições, são 100 índices, mas por isso que vai de 0 a 99 porque começa do zero. Vamos dar uma olhada em outro exemplo, pensa comigo o seguinte, a gente tem aqui uma lista de frutas, um array de frutas e o array ele tem essa característica de ter aqui posições que são os índices, na primeira posição ou no primeiro índice, que aqui é o índice 0 e aí começa de zero, a gente tem a maçã, no índice 1 o abacaxi, no índice 2 a melancia, no índice 3 a gente tem banana e no índice 4 a gente tem uva. Perceba que a gente tem 5 frutas, são 5 posições, mas a gente tem do índice 0 ao índice 4, porque são 5 posições, mas como começa do 0, então vai de 0 a 4. Agora imagina só, eu quero exibir o que tem no índice 2 desse array, eu posso usar um console.log, porque eu quero exibir isso no console, eu vou acessar aqui o meu array de fruits, que é o array, eu coloquei esse nome e para acessar um índice, eu coloco entre colchetes qual é o índice que eu quero acessar, nesse caso eu estou pedindo para acessar o índice 2, o que tem no índice 2? No 0 é a maçã, no 1 é o abacaxi, no 2 a gente tem a melancia, então o que vai aparecer é a melancia, que é o conteúdo que está ali no índice 2. Vamos olhar agora para outro exemplo, vou dar um tempo para você pensar e me responder, qual é a fruta que vai aparecer aqui agora? Eu estou pedindo para fazer um console.log, ou seja, exibir ali no console o que está no índice 0 do nosso array de frutas, o que vai aparecer agora? Se você respondeu que é maçã, acertou, porque maçã está aqui no índice 0, que é a primeira posição do nosso array. Então essa é a definição de array, uma lista onde a gente tem itens ordenados e cada item está em uma posição específica e essa posição específica a gente chama de índice. Agora que você entendeu o conceito, vamos colocar isso em prática.
