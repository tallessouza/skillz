WEBVTT Esse é o React Router que nós vamos utilizar para implementar navegação na nossa aplicação, ou seja, criar as rotas da nossa aplicação. Quando a gente utiliza o React, quando a gente cria um projeto com o React, temos apenas o React focado ali em construir as interfaces da nossa aplicação de forma declarativa e tudo mais mas a gente não tem ali por padrão essa parte de navegação, por isso a gente inclui agora o React Router, que é a estratégia que a gente utiliza e é a biblioteca mais utilizada para lidar com navegação no React, então você pode acessar reactrouter.com aqui a gente pode vir na parte de docs e aqui a gente já vai ter todas as informações para a gente utilizar ele na nossa aplicação.
