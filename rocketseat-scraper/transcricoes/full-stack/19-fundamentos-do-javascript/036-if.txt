Então, vamos lá. Deixa eu colocar como comentário aqui o IF, que é a estrutura de condição C. Imagina só que a gente tenha uma variável, que eu vou chamá-la de hour, e o valor inicial dela vai ser 11. Eu quero exibir uma mensagem de bom dia se a hora for menor ou igual a meio-dia, ou a 12. Se for menor ou igual a 12, eu quero que apareça uma mensagem de bom dia. Se não for, eu não quero que apareça nenhuma mensagem. E é aí que entra, por exemplo, a estrutura IF. O IF tem essa estrutura. A gente declara que vai ser uma condição, aqui com a palavra reservada IF, e o JavaScript sabe que dentro dessas chaves, dentro desse parênteses, na verdade, a gente vai definir uma condição e dentro dessas chaves vai os comandos, as instruções que a gente quer executar se a condição for atendida. Então, aqui eu vou pôr a condição. Se a hora for menor ou igual a 12, eu quero executar a mensagem de bom dia. E aí ele vai aparecer ali, bom dia, mas se eu trocar aqui para 13 e salvar, não aparece nada. Então, essa daqui é a estrutura do IF. Então, aqui a gente tem a condição e ele só vai executar o que tiver aqui dentro se a condição for verdadeira. A chave é utilizada para determinar o escopo do IF. Você pode colocar mais coisas aqui, por exemplo, console.log seja bem-vindo. Ele só vai mostrar tudo o que tiver aqui dentro se a condição for atendida. Se eu voltar aqui para 11, olha só, bom dia, seja bem-vindo, tá vendo? Porque aqui determina o começo e o fim do IF, essa abertura e fechamento de chaves. No entanto, a chaves é opcional quando o seu IF executa uma única instrução. Olha se eu fizer isso daqui, copio, vou duplicar aqui. Eu vou até comentar essa linha aqui, para você, daqui até aqui. Então, eu vou usar comentário mais de uma linha, barra, asterisco, asterisco, barra, para comentar esse bloquinho. Aqui está funcionando igual, certo? Mas olha só se eu tirar chaves, o que acontece? Se eu tirar chaves, continua exibindo bom dia e continua exibindo seja bem-vindo. Mas se eu aumentar aqui para 13 e essa condição não vai ser mais atendida, vai sumir as duas mensagens, certo? Vamos ver? Não, sumiu só o bom dia, o seja bem-vindo sumiu. Por quê? Quando a gente omite chaves, ele só vai pegar a próxima linha do IF como sendo parte do IF. Então, só a próxima linha ele vai executar ou não, baseado na condição. Então, só esse um cuidado que você tem que ter quando omitir chaves. Eu, particularmente, gosto mais dessa estrutura. Mesmo que eu vou executar uma linha só, eu prefiro usar chaves para ficar bem claro que o meu IF vai executar aquilo. Eu, particularmente, prefiro. Mas é bom você saber que tem essas possibilidades também.
