WEBVTT Nessa aula nós vamos falar sobre o Github. Muita gente confunde Github com Git e porque o nome é bem parecidinho mesmo, né? Mas o Github é uma plataforma online para você colocar os seus códigos através do Git. Então você vai ter um repositório Git na sua máquina, ele está ali bem legalzinho, aí o Github vem como uma solução online para linkar aquele repositório que estava na sua máquina com esse serviço da nuvem que é o Github. Existem outros também, tá pessoal? GitLab, Bitbucket, mas o Github é um dos mais comuns, um dos mais populares. E trabalhar em diversos projetos é uma grande vantagem de você ter uma nuvem dessa, podendo deixar os seus códigos. Por quê? Porque se você deixar apenas na sua máquina, fica difícil a pessoa se conectar na sua máquina. Mas quando você pega da sua máquina e coloca lá no Github, então você consegue que outras pessoas possam estar trabalhando no seu mesmo projeto. Então é colaborativo, você vai encontrar no Github muitos outros projetos, pode trabalhar com times, pode trazer outros projetos direto para a sua máquina, ok? E aí ele também vai servir como um perfil para você mostrar o seu trabalho ou como um portfólio de diversos trabalhos. Todo código que você estiver fazendo, super recomendo, tira do seu computador e joga lá no Github, deixa ele lá. E a gente vai mostrar um fluxo para você, como funciona, para você jogar esses códigos no Github, certo? Então, quando você faz isso, você consegue então mostrar o seu trabalho para outras pessoas. Outras pessoas têm acesso ao seu Github público e eles conseguem ver os trabalhos que você está fazendo. Bacana? Então nas próximas aulas a gente precisa aprender a instalar, a criar uma conta no Github, como conectar o nosso projeto, o nosso repositório com o Github, etc. Bacana?
