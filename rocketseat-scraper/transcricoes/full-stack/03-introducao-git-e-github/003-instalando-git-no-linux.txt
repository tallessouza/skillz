Eu fiz uma máquina virtual aqui para simular o ambiente Linux, que é Ubuntu, e vou seguir o passo a passo que a gente tem na documentação. Você pode então, se o seu Forfet Dora é uma outro tipo de instalação, mas para o Ubuntu Debian é esse comando que a gente vai colocar e você ainda tem um link aqui que mostra outras distros caso seja sua necessidade. Então eu vou abrir aqui o terminal, abro o terminal da minha máquina, aqui eu coloco o sudo apt-get install git. A senha da minha máquina, e aqui ele pergunta, ele já vai perguntar se ou não, nesse caso ele já foi direto e já instalou, mas algumas vezes ele vai perguntar se sim ou não para você instalar. Depois disso você pode dar uma olhadinha no seguinte, git --version, para ver se está instalado. Beleza, o Git foi instalado aqui, agora a gente tem que seguir para a nossa configuração do Git na máquina. Então, git config, traço-traço global, username, aqui eu coloco o meu nome, então eu vou colocar o meu nome e você coloca o seu nome. Aqui no git config global, user email, aqui eu coloco o meu email pessoal que está cadastrado lá no GitHub. Se você ainda não criou um GitHub, e mais tarde você vai criar, coloque o mesmo email aqui da sua máquina com o email lá do GitHub, o que a partir dessa configuração é que você depois, assim que você precisar fazer o link entre os projetos que tem na sua máquina com os projetos do GitHub, é através dessa configuração de email que ele vai observar.
