WEBVTT Para fechar esse módulo de tópicos complementares de algoritmos supervisionados, eu vou falar de Shapley Values. Apesar de Shapley Values poder ser usado em outras formas, mas eu quis trazê-lo aqui pela importância dele para a questão de interpretabilidade de modelos de Machine Learning. Então a ideia aqui, Shapley Values, que veio, na verdade, da economia, ele fornece uma maneira de quantificar a contribuição que cada variável ou cada feature tem para a previsão de um modelo, tanto no nível individual quanto agregado. Isso é bastante importante. Consigo ver no meio geral, no modelo, quanto que a variável contribuiu, mas eu consigo pegar uma instância específica e avaliar naquela instância como foi o comportamento de cada variável para fazer a previsão que ele fez. Isso é bastante interessante. Os SHAPE VALUES são calculados com base na teoria dos jogos, onde ele atribui a cada variável uma contribuição proporcional à sua importância na explicação das diferenças entre a previsão do modelo e a previsão média do conjunto de dados. Isso permite que os cientistas de dados entendam como cada variável influencia as previsões do modelo e identificar quais características são importantes para uma determinada instância de dados. Muitas vezes você encontra uma boa resposta, mas quando você começa a fazer a análise individual de algumas previsões que o modelo está fazendo, você pode encontrar algo que você ache que é inconsistente, ou que não está fazendo sentido. Então, o Shapley-Varius vai te explicar, naquela instância, qual foi a contribuição que cada variável deu. Isso vai te permitir, por exemplo, capturar algum tipo de padrão que, de forma geral, o modelo não capturou, e você poder trabalhar na melhoria desse modelo.
