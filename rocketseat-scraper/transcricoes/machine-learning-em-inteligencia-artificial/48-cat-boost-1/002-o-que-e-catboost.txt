WEBVTT O que é o CatBoost? O CatBoost é um acrônimo para Categorical Boosting, que é um algoritmo de aprendizado de máquina baseado em árvores de decisão que usa a técnica de boosting por gradiente. Lembrando que esse é um algoritmo de ensemble. Ele foi desenvolvido pela Yandex, que é uma das maiores empresas de tecnologia da Rússia, com o objetivo de oferecer uma solução robusta para problemas envolvendo dados categóricos. Quais as principais características do CatBoost? A primeira delas, pelo próprio nome, ela trabalha muito bem com dados categóricos. Mas o que significa isso? É que o CatBoost processa as colunas categóricas de forma nativa. Tem que haja uma necessidade de pré-processamento complexo como o One Hot Encoding. A gente fez isso muitas vezes, pegar uma feature que é categórica e transformar ela em uma feature numérica para poder passar ela pelo algoritmo. Nesse caso, o CatBoost já faz esse processamento das colunas já dentro do próprio algoritmo. Uma outra característica importante é a redução do overfitting. Ele faz isso utilizando técnicas avançadas para minimizar como método de ordenação e o shrink case. E a gente vai falar melhor disso mais para frente. Terceira característica importante é o treinamento eficiente.
