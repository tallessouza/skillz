WEBVTT Fala Dev, tudo bem? Eu vou explicar nesse módulo o que é a análise exploratória de dados ou no inglês exploratory data analysis. Análise exploratória de dados é um processo sistemático que é usado em projetos de ciência de análise de dados para entender e resumir as características fundamentais de um conjunto de dados. A primeira etapa da análise exploratória de dados consiste em coletar e preparar os dados que serão usados nas análises. Isso envolve a gente capturar dados de bancos de dados, de arquivos ou de outras fontes de nosso interesse e preparar esses dados para que eles estejam em conformidade com a análise que vai ser feita. Essa preparação envolve transformação de dados, entendimento das estruturas, muitas vezes unificação de dados e até tratamento de dados faltantes, que é algo que aparece aqui nesse diagrama, que é o que a todo momento da análise exploratória de dados a gente tem que estar analisando ter uma forma de tratar isso, tanto os valores ausentes quanto os outliers ou os valores atípicos. Após a coleta e a preparação de dados, a gente tem condições de começar a formular hipóteses, ou seja, a gente começa a fazer perguntas que a gente gostaria de responder com a nossa análise exploratória. Para que a gente possa comprovar ou refutar essas hipóteses que foram formuladas, a gente tem algumas técnicas de análise que são usadas nesse processo de análise exploratória de dados. A primeira delas é a análise univariada, que é a análise onde a gente pega apenas uma variável e analisa sua distribuição, extrai medidas de posição ou algumas outras medidas estatísticas para analisar o comportamento dessa variável de forma isolada. Depois a gente pode usar análise bivariada quando a gente começa a correlacionar variáveis dentro do nosso conjunto de dados. Começa a entender se uma variável está altamente correlacionada com a outra, se ela está correlacionada com o objetivo ou com aquilo que a gente quer prever no nosso modelo. Então a gente usa uma análise bivariada. A análise multivariada envolve mais de duas variáveis e geralmente está ligada a algoritmos de Machine Learning para que a gente consiga extrair essa análise multivariada, o que não será o foco desse módulo introdutório. E também temos a análise temporal, quando a gente precisa fazer uma análise de dados em sequência do tempo. Por exemplo, quero entender qual o comportamento de o preço da gasolina ao longo do tempo. Existem técnicas específicas para que a gente possa compor esse tipo de análise temporal. Um aspecto muito importante desse processo de análise exploratória de dados, não é a análise propriamente dita, mas a forma como você comunica o resultado da sua análise. Então é bastante importante que o cientista de dados ou analista de dados, ou profissional de A, nesse caso, tenha a plena consciência de como é o seu público para que ele possa comunicar os dados da melhor maneira possível, tirando qualquer tecnicismo ou qualquer linguajar que eventualmente não caiba ou que o usuário lá do outro lado não vai entender. Então essa parte de comunicação do resultado é bastante importante. Você pode criar todo um arcabouço de análises que muitas vezes se elas não forem entendidas elas não vão produzir o resultado que você deseja. Outra coisa importante que eu gostaria de explicar neste módulo é o que é a Biblioteca Pandas. A Biblioteca Pandas é uma biblioteca que a gente vai usar neste módulo e vamos usar daqui pra frente com muita frequência. É uma biblioteca de Python que ela é bastante usada para análise de dados. É a principal biblioteca de análise de dados resistente. Qual que é a principal vantagem dela? Você consegue fazer todo o processo de análise. Você consegue manipular o dado, carregar o dado de algum arquivo ou de alguma fonte de dados, limpar esse dado e analisar esse dado de forma muito eficiente. Ele fornece algumas estruturas de dados flexíveis como séries, e eu vou explicar mais pra frente o que são as séries, e os dataframes. e que permite organizar os dados em tabelas e assim realizar operações complexas como filtros e agregações para facilitar a visualização dos resultados das suas análises. O Pandas também é compatível com algumas fontes de dados. Por exemplo, se eu tiver um dado em um banco de dados, eu consigo capturar esse dado e trazer para o Pandas. Se eu tenho dado no arquivo Excel ou no arquivo CSV, eu consigo trazer esse dado para dentro do Pandas. Então, é uma ferramenta essencial para os cientistas de dados e para os analistas que exigem explorar extraíndo sites de dados de maneira eficaz e intuitiva.
