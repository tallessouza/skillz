WEBVTT Vamos começar o nosso módulo de análise exploratória de dados com pandas, onde a gente vai começar a extrair os nossos principais insights dos dados que a gente precisa trabalhar. Qual a função do EDA, ou Exploratory Data Analysis? É que a gente possa dar uma boa espiada nos dados, fazendo uma analogia, antes que a gente comece a fazer coisas mais complicadas. É como um investigador curioso que ele olha primeiro para entender do que se trata aquele problema através dos dados que ele possui. Essa análise ajuda a descobrir alguns segredos escondidos nos números, nos padrões e até alguns erros nesses dados, para que a gente possa tomar decisões mais inteligentes e contar histórias mais interessantes com os nossos dados. É como a primeira pista em um grande quebra-cabeça de informações. Em termos de assuntos, a gente vai abordar inicialmente uma parte conceitual do que é EDA e o que é a Biblioteca Pandas, que a gente vai usar para os nossos experimentos, para as nossas análises. Depois, como funciona o processo de coleta e preparação de dados, como a gente lida com dados ausentes, como é que a gente formula hipóteses. Vamos falar de análise univariada e análise bivariada. como que a gente lida com outliers e finalmente como a gente pode automatizar uma boa parte desse processo de análise exploratória de dados.
