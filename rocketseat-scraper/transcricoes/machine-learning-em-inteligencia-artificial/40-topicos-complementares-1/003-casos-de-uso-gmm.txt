WEBVTT Tratando agora de alguns casos de uso onde a gente pode usar o GMM, o primeiro deles que é o que vai ser o nosso exemplo, o nosso projeto prático dessa parte, é de agrupamento de dados ou clustering, eles são usados para agrupar dados em subconjuntos homogêneos onde cada grupo é modelado por uma distribuição gaussiana diferente. Um outro caso de uso é a modelagem de distribuições complexas. A gente utiliza os GMMs para modelar distribuições de dados que são multimodais ou que tem formas complexas que não podem ser muito bem representadas por uma única distribuição gaussiana ou distribuição normal. Um outro exemplo de uso de GMM, ele é na segmentação de imagens. Então, nas imagens, os GMMs são aplicados para segmentar uma imagem de diferentes regiões com características semelhantes. Outro caso de uso, reconhecimento de padrões e de fala. Os GMMs são usados em reconhecimento de fala para modelar as características acústicas de diferentes fonemas. Por último, a gente ainda pode usar o GMM para fazer detecção de anomalias. Então, o GMM pode ser utilizado para identificar comportamentos anômalos ou raros em conjuntos de dados, comparando as observações com a distribuição modelada.
