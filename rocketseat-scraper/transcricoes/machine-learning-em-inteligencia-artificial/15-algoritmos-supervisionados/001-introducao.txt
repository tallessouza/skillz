WEBVTT Fala Dev! A gente vai começar agora a desenvolver projetos levando em conta os algoritmos supervisionados. E o primeiro deles que a gente vai tratar nesse módulo é a regressão linear simples. Lembrando que os algoritmos supervisionados são aqueles onde a gente sabe a verdade, portanto a gente tem como confrontar a nossa predição contra aquilo que é a realidade. A gente separa entre conjunto de teste e treinamento, a gente vai ver isso com mais detalhes. E a regressão é um tipo de algoritmo que a gente usa para prever valores, tanto valores discretos quanto valores contínuos. E o que nós vamos ver nesse modo? O objetivo do módulo é apresentar conceitualmente os principais algoritmos de regressão para que possamos desenvolver projetos de machine learning que fazem previsões de valores. E faremos um projeto explorando o primeiro desses algoritmos, que é o de regressão linear simples. Nesse projeto, a gente vai fazer o processo completo, desde a captura dos dados, a análise exploratória, até a entrega do modelo através de uma API para que ela possa ser consumida por outras aplicações. Em termos de tópicos, a gente vai falar o que é análise de regressão, uma descrição conceitual, depois fazer um passeio pelos principais tipos de algoritmos de regressão, e depois a gente vai focar muito na parte prática, nesse projeto de uma regressão linear simples, que é o foco desse módulo.
