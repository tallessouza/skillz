WEBVTT As medidas de dispersão são medidas que nos permitem entender qual o grau de variação que os dados possuem e são muito úteis para a gente avaliar a consistência ou a variabilidade dos valores dentro do nosso conjunto de dados. São as três principais medidas de dispersão, a variância, o desvio padrão e o coeficiente de variação. A variança é a média dos quadrados das diferenças entre cada valor e a média aritmética. Ela fornece uma ideia de quão distantes os valores estão na média, considerando o peso de cada diferença ao quadrado. A variância não apresenta os dados na mesma unidade de medida do seu conjunto de dados, mas a variância é bastante importante para alguns outros cálculos e conceitos estatísticos, como covariância, que vão servir para modelos de machine learning. Por isso, ela aparece aqui para nós como uma importante medida de expressão. Já o desvio padrão é a raiz quadrada da variância. Ela expressa a dispersão no mesmo conceito que a variança, porém, na mesma unidade de medida dos nossos dados. Ela acaba sendo uma medida de dispersão mais comum para o uso no dia a dia, principalmente quando você precisa de uma interpretação melhor dos dados ou a interpretação melhor da variação dos dados. Você tem uma terceira medida chamada coeficiente de variação. Ela é calculada através do desvio padrão dividido pela média, ele é expresso por porcentagem. Ele indica a variabilidade relativa dos dados. Ela é bastante importante quando você quer comparar a dispersão entre conjuntos de dados diferentes. Então, se você tem dois conjuntos de dados que estão em escalas totalmente diferentes e você quer saber qual desses conjuntos estão com uma variação maior ou menor, você coloca todos eles em uma escala percentual através do coeficiente de variação. Dando um exemplo das medidas de expressão e pegando um conjunto de dados, o mesmo conjunto de dados de idade, a gente tem uma variância calculada. Então, a gente calcula a média com base na diferença de cada ponto de dados de idade. E a média ao quadrado, a gente tem a variância. Com base nessa variância, a gente extrai a raiz quadrada e tem o desvio padrão. Entendam que o desvio padrão, sim, está dentro de uma unidade de medida desse conjunto, então ele está mostrando aqui que o desvio padrão é de 22,58 anos, esse é o desvio padrão desse conjunto de dados, e ele tem um coeficiente de variação de 54,05, ou seja, se a gente pegar a média e o desvio padrão, a gente divide um pelo outro e a gente tem que a variação desse conjunto é de 54,05%. Para a gente demonstrar esses conceitos agora a gente vai para o código.
