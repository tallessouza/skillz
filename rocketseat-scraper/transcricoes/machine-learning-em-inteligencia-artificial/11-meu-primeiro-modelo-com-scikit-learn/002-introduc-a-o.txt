WEBVTT Fala, Dev. Chegou a hora de colocar a mão na massa. Nesse módulo, a gente vai desenvolver nosso primeiro modelo de machine learning com o objetivo de reforçar alguns conceitos que a gente apresentou no módulo anterior de fundamentos de machine learning. Além disso, a gente vai fazer uma exploração inicial de como funciona a biblioteca Scikit-Learn, que é uma biblioteca de aprendizado de máquina que vai nos acompanhar em boa parte da trilha. Aqui são os tópicos que a gente vai ver nesse módulo. que é a Biblioteca Site Learning, qual o problema que nós vamos resolver, qual o problema que nós vamos criar um modelo. Dentro desse processo prático, a gente vai fazer a carga de dados, vai fazer uma engenharia de features, vamos visualizar os dados, vou apresentar algumas formas diferentes de visualizar esses dados, como a gente vai separar os dados de treino e os dados de teste, como a gente vai treinar o modelo, validar o modelo e visualizar algumas métricas que a gente vai extrair desse treinamento.
