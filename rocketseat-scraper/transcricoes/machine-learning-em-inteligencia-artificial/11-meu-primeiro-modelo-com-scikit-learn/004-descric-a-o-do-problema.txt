Eu tenho um laboratório de análises clínicas que, com base nos dados de pacientes que realizaram um específico tipo de exame, que é o exame de diabetes, esse laboratório gostaria de prever, com base em algumas características desses pacientes, como idade, peso e altura, qual o resultado esperado do exame para novos pacientes. Então, a ideia aqui é a gente coletar dados de exames já realizados, onde a gente já tem as características desses pacientes e o resultado obtido desse teste desses pacientes, de forma a aprender um padrão, criar um modelo, E amanhã, se tiver novos pacientes chegando para fazer esse exame, eu coletar esses dados de entrada e tentar prever qual vai ser o resultado desse exame. Então, para isso, a gente vai treinar um algoritmo supervisionado. Revisando, ele é supervisionado porque a gente tem os dados reais dos resultados para validar o nosso modelo. Com isso, a gente vai criar um modelo preditivo que atenda à necessidade desse laboratório. Então, daqui pra frente, durante todo o módulo, a gente vai fazer todo o processo no código. Então, agora é a hora de codar.
