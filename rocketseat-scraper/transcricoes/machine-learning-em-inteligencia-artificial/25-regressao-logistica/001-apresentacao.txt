WEBVTT Fala Dev, nesse módulo a gente vai ver agora a regressão logística mais um algoritmo supervisionado. Esse algoritmo apesar de ser de regressão ele é usado para classificações binárias e multiclasse e além de fornecer a classificação da instância do objeto que você deseja ele também traz a probabilidade que é uma propriedade bastante interessante. Da mesma forma que nos outros módulos, a gente vai trazer os conceitos da regressão logística, vamos falar de algumas métricas de classificação, especificamente a curva ROC e a área sobre a curva ROC, e vamos também trabalhar no projeto prático algumas técnicas diferentes, alguns testes estatísticos para comparar médias entre grupos, e distribuir a nossa aplicação com uma API, mas não uma API qualquer, mas uma API que funcione no ambiente Docker, tirando aquela máxima de que a gente só faz as coisas para funcionarem nas nossas máquinas. Então, dessa vez a gente vai criar um componente que vai funcionar em qualquer instância de Docker. Então, espero que você aproveite bastante esse módulo e vejo vocês nos vídeos.
