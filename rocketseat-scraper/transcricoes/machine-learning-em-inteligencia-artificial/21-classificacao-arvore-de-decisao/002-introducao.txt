WEBVTT Olá, Dev. Vamos começar agora um novo módulo dentro do tópico de algoritmos supervisionados, mas agora migrando um pouco dos algoritmos de regressão, onde a gente faz a previsão de valores, para algoritmos de classificação. E o primeiro deles é o algoritmo de árvores de decisão. Mas antes disso, se a gente falar de árvore de decisão, falar um pouco do que são os algoritmos de classificação. Então, o foco desse módulo é apresentar conceitualmente esses principais algoritmos de forma que a gente possa desenvolver projetos que façam previsões de categorias ou classes. Ou seja, ao invés de a gente estar prevendo um valor numérico, a gente vai prever uma categoria. Um exemplo disso. Ah, eu quero classificar se um email é spam ou não é spam. Eu quero verificar se um tratamento vai surtir um efeito positivo ou negativo no paciente. Ou eu quero avaliar se um determinado cliente é da categoria A, B, C ou D. Então todos esses tipos de problemas que a gente resolve com algoritmos supervisionados para detecção de classes ou categorias são resolvidos com algoritmos de classificação. E a gente vai fazer um projeto explorando o primeiro desses algoritmos, que é o algoritmo de árvore de decisão. A gente vai fazer o processo completo, como nos outros modos. A gente vai fazer o EDA, e a gente vai entregar esse modelo através de uma aplicação para inferência batch, que é uma coisa nova que a gente não explorou ainda, que é quando eu tenho, por exemplo, não apenas um registro para validar ou para prever, mas eu recebo um arquivo e eu quero devolver um outro arquivo com as predições. Então, a gente vai fazer isso nesse modo. E aí falando um pouco da agenda do módulo, a gente vai primeiro falar o que é um algoritmo de classificação, dar um detalhamento um pouco maior, fazer um passeio pelos algoritmos de classificação, o que é árvore de decisão, e aí vou falar um pouco sobre o projeto de árvore de decisão, depois partindo pro projeto prático.
