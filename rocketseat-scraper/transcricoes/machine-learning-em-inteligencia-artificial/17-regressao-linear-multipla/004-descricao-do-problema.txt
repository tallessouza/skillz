WEBVTT Falando agora sobre o nosso projeto, a nossa parte técnica desse módulo, a ideia aqui é desenvolver um modelo para um laboratório de análises clínicas e ele conduz uma série de testes. Um dos testes que ele conduz é um teste de colesterol. Qual é a ideia desse laboratório? Ele quer avaliar E com base nas características dos pacientes que chegam lá para fazer o exame, algumas características como a idade, o nível de atividade física, o peso e algumas outras características, se essas variáveis podem influenciar no resultado do exame de colesterol total. Lembrando que o colesterol, só para dar uma visão técnica, você tem o colesterol bom e o colesterol ruim. A ideia é sempre maximizar o colesterol bom e minimizar o ruim. Mas aqui, o nosso objetivo, a gente não vai trabalhar com essas duas variáveis de forma independente. A gente tem uma variável só, que é o colesterol total medido nesse exame. Dessa forma, a gente vai treinar um algoritmo de regressão linear múltipla, de forma que seja possível prever o resultado do exame de colesterol dado as características do paciente. E qual vai ser o processo que a gente vai fazer? Esse processo é bem análogo ao que a gente fez no módulo anterior de regressão linear simples. A gente vai carregar os dados. Nós vamos realizar uma análise exploratória, porém vamos trabalhar algumas outras formas dessa análise exploratória, dado que agora nós temos mais de uma variável independente. Vamos preparar o dataset para treino e teste. Vamos fazer o treinamento do modelo. Vamos validar o modelo através de métricas e da análise de resíduos. Vamos salvar esse modelo para que a gente possa aproveitar esse modelo e entregar ele através de uma UI, de uma aplicação que o nosso usuário final poderia trabalhar, poderia testar. Então, agora vamos para o código.
