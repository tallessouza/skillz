WEBVTT Então, fala Dev, vamos dar início a mais um módulo da nossa trilha de API produtos. E a ideia aqui é a gente trabalhar de forma progressiva, sempre pegando o aprendizado do módulo anterior e indo além. No módulo anterior a gente viu a regressão linear simples e agora a gente vai trabalhar com regressão linear múltipla, onde teremos muitas variáveis para prever algum tipo de valor. Então, o desafio é ainda maior. Nós precisamos entender a correlação entre essas variáveis, entender qual é o conceito da regressão linear múltipla, e também vamos ter uma forma de entregar agora essa solução, esse modelo, através de uma aplicação, para que o nosso usuário possa testar o nosso modelo e validar se ele está funcionando para o problema dele. Então, espero que você curta bastante esse módulo e vejo você nos vídeos.
