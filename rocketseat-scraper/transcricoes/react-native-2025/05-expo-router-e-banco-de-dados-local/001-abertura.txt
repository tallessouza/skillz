WEBVTT Estamos dando início a um novo módulo, e um módulo incrível, tá? Nesse módulo a gente vai desenvolver do zero uma aplicação de gestão de metas financeiras. Nesse aplicativo o usuário poderá criar uma meta, definir o valor que ele pretende alcançar naquela meta, por exemplo, fazer uma viagem ou comprar alguma coisa específica, e ele pode ir guardando dinheiro, mas também ir retirando. E o aplicativo vai mostrando para ele o progresso daquela meta que ele deseja alcançar. Um aplicativo muito legal, uma aplicação incrível, inclusive para você usar no seu dia a dia, se você tiver interesse e também para você colocar no seu portfólio com certeza. A gente vai construir esse aplicativo do zero e vamos colocar nesse aplicativo a estratégia de navegação baseada em arquivos com o Expo Router. Aproveitando tudo que você já aprendeu de React Navigation, até porque o Expo Router é baseado no React Navigation. Você vai curtir muito, eu tenho certeza, essa estratégia de navegação baseado em arquivos você vai ver como fácil é inclusive muito mais simples de implementar no teu projeto e de quebra nessa aplicação a gente vai colocar banco de dados local então você vai aprender aqui a utilizar o SQLite inclusive vamos escrever código SQL dentro da nossa aplicação para manipular os dados ali do banco de dados local ali no nosso app e exibir essas informações ali na tela para o usuário.
