WEBVTT Sejam bem-vindos ao nosso novo módulo na nossa trilha de React Native e este módulo em especial é um módulo muito bacana onde nós iremos aprender coisas incríveis como por exemplo aplicar a arquitetura MVVM nas nossas aplicações React Native e esse tipo de conhecimento é de suma importância nós temos uma certa noção porque ele vai muito além do framework da linguagem que a gente utiliza ele é mais algo conceitual então Basicamente, o conceito que nós vamos aplicar aqui dentro do React Native também é possível nós aplicarmos tanto no React Web ou até mesmo no Swift ou no Kotlin, onde essa arquitetura é bastante utilizada. Então, com certeza vai ser algo que irá agregar muito para todos vocês como desenvolvedores. Também iremos aprender neste módulo como criar um sistema de cacheamento das nossas requisições HTTP utilizando o React Query, o Tanstec. Basicamente, vamos elevar a um outro nível, um outro patamar, aquilo que nós aprendemos no módulo anterior das requisições HTTP, utilizando apenas o Axios. Agora nós iremos englobar todas as nossas requisições em camadas de cache para que, muitas vezes, algum tipo de dado em que ele não se altera muito, algum tipo de dado que é comum ele se persistir daquela forma por muito não tem a necessidade de a gente simplesmente ficar batendo na nossa API o tempo todo. Basicamente, a gente vai salvar esse dado em memória do nosso dispositivo e quando houver a necessidade de nós buscarmos, realizarmos a requisição novamente, nós simplesmente pegamos de lá ao invés de bater na nossa API e isso daqui é muito massa. Melhora muito a experiência de usuário por não ficar tendo aqueles loadings infinitos o tempo todo. E outra coisa que nós iremos aprender, que vai ser muito bacana neste novo módulo, o gerenciamento de estado global utilizando o Zustand. Assim como no módulo passado, nós utilizamos o Context API, agora nós iremos utilizar uma outra ferramenta que está em ascensão no mercado. Para falar a verdade, já tem vários anos que o pessoal utiliza o Zustand e é impressionante como ele simplifica toda a parte de gestão de estado global da nossa aplicação, comparado com o Context ou com outras soluções do mercado, como por exemplo o próprio Redux. Como vocês podem ver, este módulo está recheado de coisa interessante e eu espero vocês na próxima aula, quando daremos início à contextualização de qual será o aplicativo que iremos desenvolver aqui.
