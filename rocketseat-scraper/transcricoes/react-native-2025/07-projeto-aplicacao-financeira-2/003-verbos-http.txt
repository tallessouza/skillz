Para darmos início, vamos entender o que são verbos HTTP. Os verbos HTTP, também chamados de métodos HTTP, são comandos que indicam a ação que um cliente, como por exemplo o navegador ou até mesmo no nosso contexto um aplicativo, deseja realizar em um recurso específico do servidor. Basicamente são instruções que nós, como clientes, mandamos para a nossa API para ficar mais fácil de identificar que tipo de ação nós iremos estar realizando com a chamada na nossa API. Temos aqui alguns exemplos dos verbos HTTP mais utilizados normalmente. E aqui ao lado temos uma print da documentação da API do nosso projeto que nós iremos estar desenvolvendo neste módulo. Basicamente são os métodos GET, POST, PUT e DELETE. Existem vários outros métodos HTTP, porém, esses daqui são os mais utilizados.
