WEBVTT Fala Dev, seja muito bem-vindo e bem-vinda ao primeiro módulo da formação de React Native. Nesse módulo aqui, você vai aprender todos os fundamentos de React Native que são importantes para você desenvolver qualquer aplicação. Então você vai aprender desde como o React Native funciona, como criar projetos com o React Native, os fluxos de trabalho, vai entender sobre componentes, propriedades, a trabalhar com estados, vai entender também toda a parte de estilização e como nós podemos definir o visual da nossa aplicação. Enfim, vai ser um módulo focado nos fundamentos, ou seja, toda a base sólida que você precisa entender e dominar para construir qualquer aplicação mobile, tanto para Android quanto para o iOS, utilizando o React Native.
