WEBVTT Fala Dev, bem-vindo a mais um projeto de Node aqui de dentro do Ignite e esse é um projeto para a gente desenvolver uma API REST, um serviço web utilizando o Node do total zero até o deploy, inclusive trabalhando com banco de dados, com testes automatizados e inclusive também automatizando o processo de deploy da nossa aplicação para cada vez que a gente altera alguma coisa no código, a gente subir automaticamente essa nova alteração que a gente fizer lá pra dentro da nossa URL de produção. E esse é um projeto fantástico, principalmente se for um dos seus primeiros contatos com o Node de forma mais aprofundada, Porque é um projeto que, como eu falei, a gente vai usar um monte de tecnologias legais, como Fastify, que é um micro framework para Node, a gente vai usar TypeScript, a gente vai trabalhar com SQL, a gente vai trabalhar com bibliotecas como Nex, que é uma biblioteca para trabalhar com banco de dados, a gente vai trabalhar com um monte de ferramenta legal. Mas também é um projeto muito legal porque a gente vai trabalhar com um monte de conceitos importantes que você vai ter que trabalhar aí no seu dia a dia. A gente vai trabalhar com testes automatizados, eu vou te explicar toda a diferença sobre os tipos de testes que a gente tem na nossa aplicação, quais que a gente vai priorizar, quais que a gente vai ter mais, quais que a gente vai ter menos. A gente vai trabalhar com banco de dados, vamos trabalhar com variáveis ambiente, vamos trabalhar com validação. A gente vai trabalhar... Meu Deus, tem tanta coisa, hein? Com deploy, como eu mesmo falei. A gente vai trabalhar com linting, que é padronizar o nosso código entre o nosso time. Cara, a gente vai criar muita coisa e vai ser muito legal desenvolver esse projeto junto com você e a gente se vê aí durante as aulas. Bora codar juntos e... Taca-lhe pau!
