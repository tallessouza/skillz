Caso você não conheça, UUID é uma sigla para Universal Unique ID ou Unique Universal ID. Não sei a ordem exatamente dessas duas primeiras palavras. Mas, basicamente, traduzindo, é um id único universal. Então, a ideia de usar esse uuid é que não interessa quantas vezes você chame essa função, ela sempre vai retornar um id único.
