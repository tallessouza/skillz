Então, por exemplo, handle create new todo. Por exemplo, aqui, ou seja, assim eu consigo identificar muito fácil e diferenciar funções que são enviadas a partir de um evento do usuário de funções que são apenas auxiliares para fazer algum tipo de cálculo, algum tipo de operação, algum tipo de chamada API ou qualquer coisa assim.
