Dando uma explicação um pouco mais detalhada, o backpropagation é o algoritmo que permite treinar redes neurais profundas ajustando os pesos de cada conexão entre os neurônios. Ele funciona como um processo de retroalimentação de erro. Primeiro a rede recebe uma entrada, realiza o cálculo direto até a saída, que é o feedforward. A saída obtida é comparada com o valor esperado e a diferença entre eles gera o erro. Em seguida, este é repropagado de volta para a rede, camada por camada, aplicando a regra da cadeia do cálculo diferencial. Assim como a gente teve revisão de álgebra linear, aqui a gente vai ver cálculo diferencial integral, onde a gente vê integrais e diferenciais. Especificamente, o que a gente usa aqui do cálculo diferencial é a regra da cadeia, que é a cálculo da derivada parcial. A gente vai entender melhor daqui a pouco. Para determinar como cada peso contribuiu para o erro final, lembra? Eu não estou falando mais de um perceptron. onde eu pego a saída desse perceptor, eu vejo a diferença, coloco o peso e atualizo o perceptor. Eu tenho uma rede para ser atualizada, eu tenho pesos de cada uma dessas camadas, então eu tenho que saber qual é o ajuste que eu tenho que fazer em cada peso de cada camada referente a cada entrada para minimizar o meu erro. É isso que o backpropagation faz, só que ele não faz isso sozinho, ele é um algoritmo que permite fazer esse retorno. Dessa forma a gente consegue calcular o gradiente, e aí vem o gradiente que é o próximo conceito, do erro em relação a cada peso, ou seja, a direção em que devemos ajustar cada conexão, ou seja, cada um dos neurônios, para reduzir o nosso erro.
