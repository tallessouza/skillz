WEBVTT Eu vou falar agora de mais algumas duas técnicas que a gente pode usar durante o loop de treinamento para melhorar nosso modelo e evitar algum tipo de problema. Uma das abordagens que a gente pode usar é o que a gente chama do Clip Gradients. que na verdade é uma técnica para evitar que os gradientes fiquem muito grandes durante o treinamento, tá? Que é o que a gente chama lá do Exploding, como é que a gente poderia resolver o Exploding Gradients. Então uma das formas que a gente pode usar é a gente normalizando os dados, a gente já fez isso. Uma outra forma, que eu vou mostrar aqui depois tecnicamente como fazer, é usando o Clipping Gradient. Então basicamente o que ele faz? Antes de atualizar os pesos da rede, os valores dos gradientes são limitados, ou cortados, para que não ultrapasse um valor máximo definido. E assim ele mantém o tamanho do passo de atualização controlado. E aí isso ajuda a prevenir o problema do Exploding Gradient. No PyTorch é feito com essas funções, o nnUtilsClipGradientNorm ou ClipGradeValue, que atuam diretamente nos gradientes do modelo, ajustando se eles estiverem muito grandes. Uma outra coisa que a gente pode fazer, também durante o loop de treinamento, que a gente chama de Early Stopping, que é uma técnica muito simples para prevenir overfitting. Qual que é a ideia? Ele vai monitorando a performance do modelo num conjunto de validação durante o treinamento. Quando essa performance para de melhorar, ela começa a piorar após um certo número de épocas, e esse certo número de épocas ele chama de patience, ou seja, ele tem a paciência de esperar algumas épocas e ver se vai melhorar ou se estabilizou aquele número. e aí o treinamento é interrompido. Então dessa forma o modelo é mantido no ponto que apresentou melhor desempenho nos dados de validação e aí com isso você economiza recursos e tempo e ao mesmo tempo que garante uma melhor generalização para novos dados. É uma técnica muito simples de implementar e muito usada para melhorar a robustez dos modelos. Então agora vamos para o código para mostrar como que a gente aplica essas duas técnicas.
