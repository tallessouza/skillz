Voltando aqui para os slides, a primeira arquitetura que faz sentido comentar aqui, é o que a gente chama de Redes Feedforward, ou Fully Connected, que é uma estrutura clássica que tem camadas densamente conectadas, como a gente pode ver aqui no desenho. Então, tem uma camada de input e várias camadas ocultas, todas elas conectadas entre si, para produzir a camada de saída. Essa é a estrutura dela. E ela é a arquitetura mais básica do que a gente chama de multilayer perceptron. A gente vai olhar mais pra frente o que é o perceptron, que é uma estrutura, uma forma de um neurônio, uma espécie de neurônio artificial, que é uma estrutura ainda mais básica. E aqui a gente tem um multi-layer perceptor, ou seja, são vários perceptores em várias camadas conectados entre si. Esse tipo de rede ainda é usado e é muito interessante para dados tabulares, não dados desestruturados, mas dados estruturados ou que tenham características mais vetorizadas. Uma outra arquitetura bastante importante de ser comentada é a arquitetura de redes convolucionais, que eu citei anteriormente na parte de História do Deep Learning e que foi popularizada por Ian LeCun, Benjo e outros pesquisadores. Elas são especializadas em dados de grid, como imagens ou áudio. Elas usam convoluções, que é uma transformação matemática, para detectar padrões locais, no caso de imagens, detectar por exemplo uma borda ou algum tipo de forma naquela imagem. Elas são ótimas em visão computacional, é uma das principais aplicações desse tipo de arquitetura. e ela também é interessante para reconhecimento de imagem, vídeo e processamento de sinais. Por exemplo, as CNNs, Convolutional Neural Networks, são padrão hoje para tarefas de classificação de imagem, detecção de objetos, reconhecimento facial, dentre outros. O que a gente pode até ver aqui, ainda não entrando muito no detalhe, é que a CNN tem dois blocos. Ela tem um bloco onde ela aprende características da imagem, ou daquilo que você está querendo classificar eventualmente. e ela tem uma segunda parte que na verdade é uma rede fully connected. Então na verdade aqui ele faz uma série de convoluções e outros processos matemáticos para tentar encontrar esses padrões dessa imagem. A partir do momento que ele encontra essas imagens ele coloca isso numa estrutura de rede. é Fully Connected, passa por uma camada que a gente chama aqui uma transformação de Softmax, aplica o algoritmo Softmax e ele consegue classificar, atribuindo probabilidades para aquela imagem. Se aquela imagem é um carro, é uma van, ou é um caminhão, ou até uma bicicleta. Então esse Softmax, não entrando muito detalhe, ele é usado para isso, ele atribui probabilidades para certas classes que você está tentando encontrar dentro de um problema de classificação. Então, essa aqui é uma estrutura de redes convolucionais. Uma outra arquitetura muito importante, principalmente nos dias de hoje, são as redes recorrentes, ou as RNNs, Recurrent Neural Networks. Elas são projetadas para dados em sequência, como o texto, a fala ou até séries temporais. Nesse caso, cada neurônio tem uma memória do estado anterior e realimenta essa informação. A saída de um passo pode influenciar o próximo, como vemos aqui no desenho.
