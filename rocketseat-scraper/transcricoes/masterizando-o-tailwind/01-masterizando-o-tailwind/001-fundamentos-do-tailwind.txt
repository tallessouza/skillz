Nessa aula a gente vai começar a falar sobre uma das ferramentas que mais transformou a maneira da gente estilizar as nossas aplicações front-end nos últimos anos, que é o Tailwind CSS. O Tailwind CSS, como a própria documentação já diz, é um Utility First CSS Framework. Ou seja, ele é um framework para CSS focado em utilities. Utilities nada mais são do que classes que aplicam alguma estilização. E veja que aqui para fazer a estilização, por exemplo, desse card aqui na esquerda, eu preciso, claro, várias tags HTML, porém várias classes onde cada uma das classes aplica um efeito CSS, uma propriedade com o valor CSS diferente aqui dentro. E o que todas essas ferramentas têm em comum? Elas usam de um conceito chamado de interface declarativa. A interface declarativa é aquela onde a gente estiliza a interface muito próximo do local onde a gente define a estrutura da nossa interface.
