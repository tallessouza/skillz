# Hair Day — Contexto do Projeto

> O projeto Hair Day é um sistema de agendamento de corte de cabelo onde HTML e CSS já estão prontos — todo o desenvolvimento é focado em adicionar funcionalidades com JavaScript.

## Key concept

Hair Day é um projeto prático que une todos os conceitos de JavaScript aprendidos anteriormente em uma aplicação real. O visual (HTML + CSS) é fornecido pronto — o foco total é na implementação das funcionalidades via JavaScript.

## Decision framework

| Quando encontrar | Aplique |
|-----------------|---------|
| Necessidade de alterar visual | Não altere HTML/CSS — eles já estão prontos, foque no JS |
| Dúvida sobre escopo | Funcionalidades de agendamento de corte de cabelo |
| Escolha de abordagem | Aplicar conceitos JS já aprendidos (DOM, eventos, fetch, etc.) |

## Escopo do projeto

- **Domínio:** Agendamento de corte de cabelo
- **Frontend:** HTML e CSS já fornecidos
- **Desenvolvimento:** Todas as funcionalidades em JavaScript puro
- **Abordagem:** Passo a passo, aula por aula

## Limitações

Este skill é apenas contexto do projeto. As funcionalidades específicas (manipulação de DOM, requisições, validações) são cobertas nas aulas subsequentes do módulo.
