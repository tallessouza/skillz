## Equacao do profissional qualificado

```
PROFISSIONAL QUALIFICADO = Hard Skills + Soft Skills + Tempo

Hard Skills (tecnicas):
  - Linguagens de programacao
  - Frameworks
  - Banco de dados
  - Logica

Soft Skills (comportamentais):
  - Comunicacao
  - Resolucao de problemas
  - Gerenciamento de tempo
  - Trabalho em equipe

Tempo:
  - ~2-3 anos de estudo consistente

"Contratado pelas hard skills, demitido pelas soft skills"
```
