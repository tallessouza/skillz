# Code Examples: Perfil de Viagens — Setup do Projeto

## Settings.json do VS Code (configuracao do instrutor)

### Configuracao minima para o projeto

```json
{
  "liveServer.settings.donotShowInfoMsg": true
}
```

### Acessar o settings.json

**Mac:**
```
Cmd+P → digitar "> Open Settings JSON" → Enter
```

**Windows/Linux:**
```
Ctrl+P → digitar "> Open Settings JSON" → Enter
```

## Atalhos importantes

### Zen Mode

| Sistema | Atalho |
|---------|--------|
| Mac | `Cmd+K` depois `Z` |
| Windows/Linux | Buscar via Command Palette: `> Toggle Zen Mode` |

### Command Palette

| Sistema | Atalho |
|---------|--------|
| Mac | `Cmd+Shift+P` ou `Cmd+P` seguido de `>` |
| Windows/Linux | `Ctrl+Shift+P` ou `Ctrl+P` seguido de `>` |

### Extensoes

| Sistema | Atalho |
|---------|--------|
| Mac | `Cmd+Shift+X` |
| Windows/Linux | `Ctrl+Shift+X` |
