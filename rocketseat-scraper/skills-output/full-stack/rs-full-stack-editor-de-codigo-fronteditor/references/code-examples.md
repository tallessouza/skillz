# Code Examples: FrontEditor.dev

## Exemplo basico demonstrado na aula

### HTML
```html
<h1>Título</h1>
```

### CSS
```css
h1 {
  color: red;
}
```

### Fluxo de uso tipico

1. Acessar `fronteditor.dev` no navegador
2. Editar o arquivo HTML com a estrutura desejada
3. Editar o arquivo CSS para estilizar
4. Observar o resultado no preview em tempo real (live reload ativado)
5. Opcionalmente editar JavaScript para adicionar interatividade
6. Usar o botao de download para salvar os arquivos

### Usando Emmet no editor

```
# Digitar e pressionar Enter:
h1          →  <h1></h1>
div.container →  <div class="container"></div>
ul>li*3     →  <ul><li></li><li></li><li></li></ul>
```

### Markdown para anotacoes

O arquivo Markdown serve para anotacoes pessoais durante o estudo. O conteudo escrito nele **nao aparece** no preview do navegador — e apenas para referencia do aluno.

```markdown
# Minhas anotacoes

- Lembrar de usar tags semanticas
- CSS Grid vs Flexbox: usar Grid para layout 2D
- Revisar seletores CSS
```
