# Personas de Dados

> Entenda as tres personas de dados (analista, cientista, engenheiro) para escolher sua carreira e colaborar com seus pares.

## Key concept

As carreiras de dados se organizam em um espectro entre **negocio** e **dado bruto**. O analista de dados fica mais proximo do negocio, o engenheiro de dados mais proximo do dado bruto, e o cientista de dados no meio. Essa nao e uma hierarquia de valor — sao especializacoes diferentes com dificuldades proprias.

## Decision framework

| Quando voce encontra | Aplique |
|---------------------|---------|
| Preciso responder "o que aconteceu?" | Analista de dados — analise diagnostica |
| Preciso prever "o que vai acontecer?" | Cientista de dados — analise preditiva |
| Preciso otimizar como os dados sao armazenados e servidos | Engenheiro de dados — infraestrutura |
| Preciso decidir "o que fazer para melhorar?" | Cientista de dados — analise prescritiva |
| Time pequeno, uma pessoa faz tudo | Provavelmente analista com skills de cientista |

## How to think about it

### Espectro negocio vs dado bruto

```
[NEGOCIO]  ◄──────────────────────►  [DADO BRUTO]
 Analista      Cientista      Engenheiro
 de Dados      de Dados       de Dados
```

O analista coleta, limpa e trata dados para responder perguntas rapidas de negocio (dashboards, analise de churn, queda de vendas). O cientista constroi modelos preditivos e prescritivos. O engenheiro estrutura e otimiza bancos de dados para que analistas e cientistas trabalhem com dados uteis.

### Ferramentas por persona

| Persona | Ferramentas principais |
|---------|----------------------|
| Analista de dados | Excel, SQL, Power BI, Python |
| Cientista de dados | Python, machine learning, estatistica avancada |
| Engenheiro de dados | SQL avancado, bancos de dados, pipelines, infraestrutura |

### Faixas salariais (Brasil, 2025)

| Persona | Junior | Senior |
|---------|--------|--------|
| Analista de dados | R$3.500–5.000 | R$8.000–12.000 |
| Cientista de dados | R$6.500–8.000 | R$14.000–18.000 |
| Engenheiro de dados | Similar ao cientista | R$17.000–18.000 |

### Transicao de carreira

A porta de entrada e quase sempre **analista de dados**. Empresas raramente contratam cientista de dados junior sem historico como analista. Estagios especificos de ciencia de dados sao raros — busque estagios de analise de dados.

## Common misconceptions

| People think | Reality |
|-------------|---------|
| Engenheiro de dados e "o novo dev" e superior | Sao especializacoes paralelas, nao hierarquia |
| As funcoes sao rigidamente separadas | Na pratica, empresas misturam escopos — analista pode fazer trabalho de cientista |
| Preciso dominar uma ferramenta especifica | Somos agnosticos de ferramenta — domine a base e se adapta a qualquer uma |
| Preciso aprender R | R e usado por empresas muito especificas; Python cobre a maioria dos cenarios |
| O executivo quer ver tudo em dashboard | As vezes ele prefere Excel para poder mexer nos dados |

## When to apply

- Ao planejar carreira em dados
- Ao montar um time de dados e definir responsabilidades
- Ao decidir quais skills priorizar no aprendizado
- Ao entender quem sao seus pares no trabalho

## Limitations

- Faixas salariais variam muito por regiao, porte da empresa e momento do mercado
- O mercado de dados ainda esta em formacao — escopos mudam rapidamente
- Agentes de IA estao criando novas responsabilidades que ainda nao tem dono definido (engenheiro ou cientista?)
- Esta formacao foca em analista de dados com introducao aos outros perfis
