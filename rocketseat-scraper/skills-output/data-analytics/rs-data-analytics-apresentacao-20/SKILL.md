# Fundamentos de Python para Data Analytics — Roadmap

> Domine os fundamentos do Python na ordem certa antes de trabalhar com dados: sintaxe basica → tipos → operadores → controle de fluxo → strings.

## Contexto

Python e uma das principais linguagens no mundo de dados. Este roadmap define a sequencia de aprendizado dos fundamentos necessarios antes de entrar em bibliotecas como pandas, numpy ou ferramentas de visualizacao.

## Sequencia de aprendizado

| Ordem | Topico | Por que importa para dados |
|-------|--------|---------------------------|
| 1 | O que e Python e onde usar | Entender o ecossistema e aplicacoes em dados |
| 2 | Comentarios e sintaxe basica | Codigo legivel e documentado |
| 3 | Variaveis e tipos de dados | Fundamento para manipulacao de dados |
| 4 | Operadores aritmeticos e logicos | Calculos e filtragens |
| 5 | Estruturas de selecao (if/elif/else) | Logica condicional em pipelines |
| 6 | Laco while (contadores, while true) | Iteracao controlada |
| 7 | Laco for (range, iteracao de strings) | Percorrer estruturas de dados |
| 8 | Metodos de string | Limpeza e transformacao de texto |

## Decision framework

| Situacao | Recomendacao |
|----------|-------------|
| Nunca programou antes | Comece pelo topico 1, siga a ordem |
| Ja programa em outra linguagem | Revise topicos 3-5, foque em 6-8 |
| Quer ir direto para pandas | Pare — domine pelo menos ate topico 7 primeiro |
| Precisa limpar dados textuais | Topico 8 (metodos de string) e pre-requisito |

## Heuristica de prontidao

Voce esta pronto para avancar para bibliotecas de dados quando conseguir:

1. Declarar variaveis de diferentes tipos sem consultar documentacao
2. Escrever um if/elif/else com operadores logicos combinados
3. Usar for com range e iterar sobre strings
4. Aplicar pelo menos 5 metodos de string (upper, lower, strip, split, replace)

## Anti-patterns

| Erro comum | Abordagem correta |
|------------|-------------------|
| Pular fundamentos e ir direto para pandas | Dominar tipos, loops e strings antes |
| Decorar sintaxe sem entender o conceito | Praticar com exercicios, entender o "por que" |
| Ignorar metodos de string | Essenciais para limpeza de dados — aprender cedo |
