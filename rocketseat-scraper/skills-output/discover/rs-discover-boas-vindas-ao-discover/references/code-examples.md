# Code Examples: Projeto Agregador de Links

## O projeto do Discover

O curso culmina na criacao de um **agregador de links** (similar ao Linktree), um projeto pratico que demonstra conceitos fundamentais de programacao web.

### Funcionalidades do projeto

1. **Agregador de links** — pagina com multiplos links organizados
2. **Light Mode / Dark Mode** — alternancia de temas visuais
3. **Troca de imagem** — avatar/foto que muda com o tema
4. **Links personalizaveis** — adicionar links de redes sociais e outros
5. **Link unico compartilhavel** — para usar em bio do Instagram

### Valor pratico

- Pode ser usado como link na bio do Instagram
- Pessoas monetizam projetos similares: R$50 a R$150
- Venda para amigos, familiares, pequenas empresas
- Funciona como primeiro item de portfolio

### Tecnologias envolvidas

O projeto usa HTML, CSS e JavaScript basico (detalhes nas proximas aulas do curso).

### Estrutura conceitual

```
agregador-de-links/
├── index.html          # Estrutura da pagina
├── style.css           # Estilos + light/dark mode
├── script.js           # Toggle de tema e imagem
└── assets/
    ├── avatar-light.png
    └── avatar-dark.png
```

### Nota sobre escopo

Este e um projeto de ~5 horas de curso, projetado para ser simples mas funcional. A simplicidade e intencional — o objetivo e dar confianca nos primeiros passos, nao construir algo complexo.
