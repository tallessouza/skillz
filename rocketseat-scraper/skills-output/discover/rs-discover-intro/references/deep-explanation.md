# Deep Explanation: Aprender a Aprender — Intro

## O insight central do Mayk Brito

Mayk usa o termo "reprogramar" de forma deliberada — não é apenas uma analogia com programação. Ele está dizendo que os alunos chegam com um "software" de estudo instalado pela escola tradicional, e esse software é incompatível com o aprendizado de programação. Antes de instalar novas técnicas, é preciso desinstalar o método antigo.

## Por que o método escolar falha em programação

O ensino tradicional otimiza para **retenção declarativa**: memorizar fatos, datas, fórmulas. A avaliação é feita por provas que testam recall. Programação exige **habilidade procedural**: resolver problemas novos aplicando padrões aprendidos em contextos diferentes.

A analogia mais precisa: ninguém aprende a nadar lendo um livro sobre natação. Programação é a mesma coisa — é uma habilidade que se desenvolve fazendo, errando e corrigindo.

## O que significa "aprendizado acelerado"

Mayk não está prometendo atalhos. "Acelerado" significa **eficiente** — extrair mais aprendizado por hora investida. Isso acontece quando:

1. **Prática ativa substitui consumo passivo** — escrever código > assistir tutorial
2. **Feedback é imediato** — o compilador/runtime te diz se funcionou em segundos
3. **Repetição é espaçada** — revisar conceitos em intervalos crescentes fixa melhor que maratona
4. **Contexto é relevante** — aprender X quando precisa de X para um projeto, não porque está no currículo

## A mentalidade como fundação

Esta aula introdutória é propositalmente curta e conceitual. Mayk está preparando o terreno: antes de ensinar técnicas específicas (que virão nas próximas aulas), ele precisa que o aluno aceite que **o jeito antigo não serve**. Sem essa aceitação, as técnicas novas serão filtradas pelo modelo mental antigo e perderão eficácia.

## Conexão com o módulo "Aprendendo a aprender"

Este é o primeiro módulo do curso Discover da Rocketseat — significativamente, não começa com código. Começa com **como aprender**. Isso reflete a filosofia da Rocketseat: a habilidade meta (aprender a aprender) é mais valiosa que qualquer habilidade técnica específica, porque habilita todas as outras.
