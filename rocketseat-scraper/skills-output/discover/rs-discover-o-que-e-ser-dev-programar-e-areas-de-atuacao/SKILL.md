# O que é ser Dev — Áreas e Habilidades

> Orientar decisoes de carreira em desenvolvimento com clareza sobre areas, habilidades e diferenciais profissionais.

## Key concept

Programar é resolver problemas do mundo real atraves de codigo. Um **programador** pega tarefas e resolve. Um **desenvolvedor** pensa sobre a tarefa, propoe a melhor solucao, debate, discute e faz levantamentos — combina hard skills com soft skills.

O mercado busca profissionais **qualificados**, nao qualquer profissional. Qualificado = hard skills + soft skills.

## Decision framework

| Quando encontrar | Aplicar |
|-----------------|---------|
| Pessoa indecisa entre areas | Apresentar: Web, Mobile, Dados, IA, IoT, Analise, Negocios — e perguntar o que atrai |
| Duvida front vs back vs full-stack | Front = visual + interacao do usuario. Back = logica + regras de negocio + banco de dados. Full-stack = ambos, mais oportunidades |
| Pessoa frustrada com dificuldade | Programacao e desafiadora por natureza — por isso o mercado paga bem e demanda profissionais |
| Pessoa sem experiencia formal | 2 anos de experiencia podem ser comprovados via projetos, comunidade e portfolio, nao apenas emprego formal |
| Pessoa focando so em codigo | Alertar: soft skills (comunicacao, resolucao de problemas, gestao de tempo, trabalho em equipe) sao tao importantes quanto hard skills |

## How to think about it

### Hard Skills vs Soft Skills

Hard skills sao habilidades tecnicas — linguagens, frameworks, conceitos. Soft skills sao comportamentais — comunicacao, resolucao de problemas, gestao de tempo, trabalho em equipe. Pessoas sao contratadas pelas hard skills e demitidas pelas soft skills.

### Programador vs Desenvolvedor

O programador executa tarefas. O desenvolvedor pensa sobre a tarefa, traz a melhor solucao, debate, discute, faz levantamentos. A diferenca esta nas soft skills e na visao ampla do problema.

### Areas de atuacao

| Area | Foco | Observacao |
|------|------|------------|
| Web | Sites, aplicacoes web | Front-end, Back-end, Full-stack |
| Mobile | Apps para celulares/tablets | iOS, Android, cross-platform |
| Dados | Analise, ciencia de dados | Exige matematica e estatistica |
| IA | Inteligencia artificial | Machine learning, deep learning |
| IoT | Internet das coisas | Dispositivos conectados (ar-condicionado Wi-Fi, robos) |
| Negocios | Tech + business | Empreendedorismo tech |
| Analista | Levantamento de requisitos | Vai alem do codigo |

### Front-end vs Back-end vs Full-stack

- **Front-end**: parte visual, interacao do usuario, o que se ve na tela
- **Back-end**: logica, regras de negocio, banco de dados, seguranca (login/senha)
- **Full-stack**: trabalha ambos, tende a ter mais oportunidades

## Common misconceptions

| Pensam | Realidade |
|--------|-----------|
| Tem muita vaga, entao e facil | Vagas exigem profissionais qualificados — programacao e desafiadora |
| Preciso de emprego formal para ter experiencia | Projetos pessoais, comunidade e portfolio comprovam experiencia |
| So preciso saber codar | Soft skills definem permanencia e crescimento na carreira |
| Se nao gostei de front-end, TI nao e pra mim | Existem muitas areas — testar back-end, dados, mobile antes de desistir |
| Full-stack e sempre melhor | Depende do objetivo — especialistas tambem sao valorizados |

## When to apply

- Orientando alguem que esta comecando em programacao
- Explicando diferencas entre areas de desenvolvimento
- Ajudando na decisao de foco de carreira
- Discutindo o que o mercado espera de um profissional

## Limitations

- Nao substitui mentoria personalizada de carreira
- Salarios e demanda variam por regiao e momento do mercado
- Este skill nao ensina a programar — orienta decisoes sobre a carreira
