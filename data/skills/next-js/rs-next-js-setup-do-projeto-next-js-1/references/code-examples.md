# Code Examples: Setup do Projeto Next.js (App Router)

## 1. Comando de criacao do projeto

```bash
# O instrutor usa pnpm especificamente
pnpm create next-app --empty board
```

Opcoes selecionadas durante o setup interativo:
- TypeScript: **Yes**
- Biome: **Yes** (ao inves de ESLint)
- React Compiler: **Yes**
- Tailwind CSS: **Yes**
- src/ directory: **Yes**
- App Router: **Yes**

## 2. next.config.ts resultante

```typescript
// O config vem "bem cru" — apenas React Compiler habilitado
import type { NextConfig } from "next"

const nextConfig: NextConfig = {
  experimental: {
    reactCompiler: true,
  },
}

export default nextConfig
```

## 3. src/app/page.tsx (template empty)

```typescript
// Apenas "Hello World" — sem estilos, sem SVGs, sem boilerplate
export default function Page() {
  return <div>Hello World</div>
}
```

## 4. src/app/layout.tsx

```typescript
import type { Metadata } from "next"
import "./globals.css"

// Metadata substitui o <head> — Next gera automaticamente
export const metadata: Metadata = {
  title: "Create Next App",
  description: "Generated by create next app",
}

export default function RootLayout({
  children,
}: Readonly<{
  children: React.ReactNode
}>) {
  return (
    <html lang="en">
      <body>{children}</body>
    </html>
  )
}
```

**Observacao do instrutor:** "Veja que ele nao tem o head, porque o head, na verdade, e criado pelo Next de forma automatizada com base nesse metadata."

## 7. biome.json — configuracao personalizada

```json
{
  "$schema": "https://biomejs.dev/schemas/1.9.4/schema.json",
  "vcs": {
    "enabled": false,
    "clientKind": "git",
    "useIgnoreFile": false,
    "defaultBranch": "main"
  },
  "organizeImports": {
    "enabled": true
  },
  "linter": {
    "enabled": true,
    "rules": {
      "recommended": true
    }
  },
  "javascript": {
    "formatter": {
      "semicolons": "asNeeded"
    }
  }
}
```

O instrutor demonstra ao vivo:
- Espacos extras no save → Biome remove automaticamente
- Virgula faltando → Biome adiciona automaticamente
- `semicolons: "asNeeded"` → remove ponto e virgula opcionais
