# Code Examples: Metadados de Paginas no Next.js

## Exemplo 1: Layout raiz basico (antes)

O ponto de partida — o que o `create-next-app` gera por padrao:

```typescript
// app/layout.tsx
import type { Metadata } from "next";

export const metadata: Metadata = {
  title: "Create Next App",
  description: "Generated by create next app",
};

export default function RootLayout({
  children,
}: {
  children: React.ReactNode;
}) {
  return (
    <html lang="en">
      <body>{children}</body>
    </html>
  );
}
```

Resultado na aba: "Create Next App"

## Exemplo 3: Template com placeholder %s

O instrutor evolui para o padrao com template:

```typescript
// app/layout.tsx
export const metadata: Metadata = {
  title: {
    template: "%s | Product Roadmap",
    default: "Product Roadmap",
  },
  description: "Follow the development progress of our entire platform.",
};
```

- Se uma pagina exporta `title: "Board"` → aba mostra "Board | Product Roadmap"
- Se uma pagina NAO exporta title → aba mostra "Product Roadmap" (default)

## Exemplo 5: generateMetadata para paginas dinamicas

O instrutor mostra o caso de uso para paginas de produto:

```typescript
// app/products/[id]/page.tsx
import type { Metadata } from "next";

export async function generateMetadata({
  params,
}: {
  params: { id: string };
}): Promise<Metadata> {
  // Aqui dentro pode fazer requisicoes HTTP
  const product = await fetch(
    `https://api.example.com/products/${params.id}`
  ).then((res) => res.json());

  return {
    title: product.name,
    description: product.description,
  };
}

export default function ProductPage({
  params,
}: {
  params: { id: string };
}) {
  return <div>Product {params.id}</div>;
}
```

Resultado: "Nome do Produto | Product Roadmap"

## Exemplo 6: Metadata completo com OpenGraph e Twitter

Expansao do que o instrutor menciona ser possivel:

```typescript
// app/layout.tsx
import type { Metadata } from "next";

export const metadata: Metadata = {
  title: {
    template: "%s | Product Roadmap",
    default: "Product Roadmap",
  },
  description: "Follow the development progress of our entire platform.",
  openGraph: {
    title: "Product Roadmap",
    description: "Track our platform development",
    type: "website",
  },
  twitter: {
    card: "summary_large_image",
    title: "Product Roadmap",
  },
  robots: {
    index: true,
    follow: true,
  },
};
```

## Variacao: Separador diferente no template

```typescript
// Com ponto
title: { template: "%s · Product Roadmap", default: "Product Roadmap" }

// Com hifen
title: { template: "%s - Product Roadmap", default: "Product Roadmap" }

// Com barra
title: { template: "%s / Product Roadmap", default: "Product Roadmap" }
```

O instrutor usa `|` (barrinha) como exemplo, mas qualquer separador funciona.
