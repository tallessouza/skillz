# Code Examples: Encerramento — Iniciando com Angular

## Nota

Esta aula de encerramento nao contem exemplos de codigo — e uma reflexao sobre o modulo completo. Os exemplos praticos estao nas aulas anteriores do modulo "Iniciando com Angular".

## Exercicio sugerido para consolidacao

O instrutor sugere revisitar e praticar. Um exercicio de consolidacao seria:

```bash
# Criar um projeto Angular do zero
ng new meu-projeto-pratica --standalone

# Entrar no projeto
cd meu-projeto-pratica

# Iniciar o servidor de desenvolvimento
ng serve
```

Apos criar, tente sem consultar material:
1. Criar um novo componente
2. Adicionar routing basico
3. Exibir dados dinamicos no template
4. Criar um service simples

Se travar em algum passo, esse e exatamente o ponto que precisa ser revisitado no modulo.
