# O Proximo Passo da Jornada Dev

> Apos dominar fundamentos (HTML, CSS, JS, Git), o proximo passo e aprofundar em tecnologias de mercado com projetos praticos e preparacao comportamental.

## Progressao de aprendizado

| Nivel | Conteudo | Resultado |
|-------|----------|-----------|
| Discover (completo) | HTML, CSS, JS, Git fundamentos | 1 projeto, certificado |
| Explorer (proximo) | Node, React, Backend, habilidades comportamentais | 10+ projetos, preparacao para entrevistas |
| Profissional | Especializacao, projetos reais | Vagas, freelance, empresa propria |

## Decision framework

| Situacao | Recomendacao |
|----------|-------------|
| Terminou fundamentos, quer emprego | Aprofundar em Node + React + Backend, construir portfolio com 10+ projetos |
| Quer freelance/negocio proprio | Focar em stack completa (front + back), monetizar projetos |
| Inseguro sobre programacao | Continuar praticando com desafios, programacao e aprendizado continuo |
| Quer se destacar em entrevistas | Desenvolver habilidades comportamentais alem do tecnico |

## Tecnologias demandadas pelo mercado

- **Frontend:** React
- **Backend:** Node.js
- **Fundamentos:** HTML, CSS, JavaScript
- **Ferramentas:** Git, GitHub

## Pilares para primeira vaga

1. **Qualificacao tecnica** — dominar tecnologias que as vagas pedem
2. **Portfolio de projetos** — minimo 10 projetos praticos demonstraveis
3. **Habilidades comportamentais** — saber se comunicar em entrevistas
4. **Aprendizado continuo** — programacao nunca tem ponto final, sempre ha proximo passo
5. **Presenca profissional** — certificados no LinkedIn, projetos visiveis

## Realidade do mercado

- Mercado tem centenas de milhares de vagas (500mil+ no Brasil)
- Vagas sao para profissionais **qualificados** — nao basta saber o basico
- Possibilidades: trabalho remoto, freelance, empresa propria, CLT
- Diferencial: quem estuda continuamente e constroi projetos reais
