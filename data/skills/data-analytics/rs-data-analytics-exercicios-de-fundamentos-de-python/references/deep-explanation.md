# Deep Explanation: Exercicios de Fundamentos de Python

## Por que praticar com notebooks

O instrutor enfatiza que a pratica e essencial apos aprender os fundamentos. O formato de notebook Jupyter e ideal porque:

1. **Interatividade** — cada questao tem uma celula de codigo logo abaixo para resposta imediata
2. **Feedback instantaneo** — executar a celula mostra o resultado na hora
3. **Contexto preservado** — a questao e a resposta ficam juntas, facilitando revisao

## Google Colab como ambiente padrao

O Colab e recomendado como primeiro ambiente porque:
- Nao requer instalacao local
- Ja tem Python e bibliotecas de dados pre-instaladas
- Funciona em qualquer navegador
- Gratuito com GPU disponivel

### Fluxo de upload no Colab

1. No repositorio GitHub, clicar no icone de download do notebook
2. Abrir colab.research.google.com
3. Fazer upload do arquivo `.ipynb`
4. O Colab renderiza as celulas markdown (questoes) e celulas de codigo (respostas)

## Alternativas locais

O instrutor menciona que VS Code e Cursor tambem executam notebooks, desde que:
- A extensao Jupyter esteja instalada e habilitada
- Um ambiente Python esteja configurado no sistema
- O kernel correto esteja selecionado

Isso e util para quem prefere trabalhar offline ou ja tem um ambiente de desenvolvimento configurado.

## Estrategia de estudo recomendada

1. Tentar resolver cada exercicio sozinho primeiro
2. Executar e verificar o resultado
3. Somente depois consultar o gabarito
4. Se errou, entender a diferenca entre sua solucao e o gabarito
