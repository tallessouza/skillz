# Code Examples: IA como Ferramenta de Aprendizado

## Nota

Esta aula é um encerramento de módulo sem exemplos de código. Os exemplos práticos de uso de IA para aprendizado foram cobertos nas aulas anteriores do módulo "IA no Aprendizado".

## Exemplos de prompts para aprendizado (derivados do contexto do módulo)

### Pedir explicação conceitual

```
Prompt: "Explique o conceito de closures em JavaScript como se eu fosse
um iniciante. Use uma analogia do mundo real."
```

### Decompor problema complexo

```
Prompt: "Estou tentando implementar autenticação JWT em uma API Node.js.
Quebre isso em passos menores e explique cada um antes de mostrar código."
```

### Gerar exercícios práticos

```
Prompt: "Crie 5 exercícios progressivos sobre array methods em JavaScript
(map, filter, reduce). Comece simples e aumente a dificuldade."
```

### Revisar entendimento

```
Prompt: "Vou explicar como entendo o event loop do Node.js. Me corrija
se eu estiver errado: [sua explicação]"
```

Esses padrões de prompt refletem a mentalidade ensinada no módulo: usar IA como parceira de aprendizado, não como atalho.
