# Code Examples: Mapas Mentais com Whimsical

## Nota: Esta aula nao envolve codigo

Esta aula e sobre uma ferramenta visual (Whimsical), nao sobre programacao. Os "exemplos" aqui sao estruturas de mapas mentais que o instrutor demonstra ou sugere.

## Estrutura demonstrada pelo instrutor

```
HTML (centro)
├── Hypertext
├── Markup
└── Language
```

## Dica pratica

Ao assistir cada modulo do curso, crie um mapa mental correspondente no Whimsical. No final do curso, voce tera uma biblioteca visual de todo o conhecimento adquirido — util para revisao rapida e para identificar gaps no aprendizado.
