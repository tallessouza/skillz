# Code Examples: tldraw

## Acesso

Nao ha codigo envolvido — tldraw e uma ferramenta web acessada diretamente:

```
URL: https://www.tldraw.com/
```

## Cenarios de uso pratico

### Cenario 1: Estudando uma API REST

Abra tldraw e desenhe:
- Caixas representando Client, Server, Database
- Setas com os metodos HTTP (GET, POST, PUT, DELETE)
- Anotacoes sobre o que cada endpoint faz

### Cenario 2: Entendendo o fluxo de um componente React

Desenhe:
- O componente pai e os filhos
- As props passadas entre eles
- O fluxo de estado (useState, useEffect)

### Cenario 3: Planejando a estrutura de um banco de dados

Desenhe:
- Tabelas como caixas com campos listados
- Linhas conectando relacionamentos (1:N, N:N)
- Anotacoes sobre indices e constraints
