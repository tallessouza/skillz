# Code Examples: Instalacao e Configuracao do Prisma

## Instalacao completa passo a passo

### 1. Instalacao do pacote

```bash
# Versao especifica (recomendado pelo instrutor)
npm install prisma@5.19.1

# Verificar instalacao
npx prisma --version
```

### 2. Inicializacao com PostgreSQL

```bash
npx prisma init --datasource-provider postgresql
```

Saida esperada:
```
✔ Your Prisma schema was created at prisma/schema.prisma
  You can now open it in your favorite editor.

Next steps:
1. Set the DATABASE_URL in the .env file to point to your existing database.
2. Run prisma db pull to turn your database schema into a Prisma schema.
3. Run prisma generate to generate the Prisma Client.
```

### 3. Arquivo .env gerado (antes da configuracao)

```env
# This was generated by `prisma init`:
# Environment variables declared in this file are automatically made available to Prisma.

DATABASE_URL="postgresql://johndoe:randompassword@localhost:5432/mydb?schema=public"
```

### 4. Arquivo .env configurado (apos configuracao)

```env
DATABASE_URL="postgresql://postgres:postgres@localhost:5432/api?schema=public"
```

### 5. schema.prisma limpo (sem comentarios)

```prisma
generator client {
  provider = "prisma-client-js"
}

datasource db {
  provider = "postgresql"
  url      = env("DATABASE_URL")
}
```

## Variacoes de DATABASE_URL por ambiente

### Desenvolvimento local (padrao do instrutor)

```env
DATABASE_URL="postgresql://postgres:postgres@localhost:5432/api?schema=public"
```

### Desenvolvimento com porta customizada

```env
DATABASE_URL="postgresql://postgres:postgres@localhost:5032/api?schema=public"
```

Nota: o instrutor mencionou usar porta 5032 no seu setup.

### Producao (exemplo com Supabase)

```env
DATABASE_URL="postgresql://postgres:SenhaSegura123@db.abcdefgh.supabase.co:5432/postgres?schema=public"
```

### Producao (exemplo com Railway)

```env
DATABASE_URL="postgresql://postgres:senha@containers-us-west-1.railway.app:6543/railway?schema=public"
```

### Docker local

```env
DATABASE_URL="postgresql://postgres:postgres@localhost:5432/api?schema=public"
```

Compose correspondente:

```yaml
services:
  postgres:
    image: postgres:16
    environment:
      POSTGRES_USER: postgres
      POSTGRES_PASSWORD: postgres
      POSTGRES_DB: api
    ports:
      - "5432:5432"
```

## Inicializacao com outros providers (para referencia)

```bash
# MySQL
npx prisma init --datasource-provider mysql

# SQLite
npx prisma init --datasource-provider sqlite

# SQL Server
npx prisma init --datasource-provider sqlserver

# MongoDB
npx prisma init --datasource-provider mongodb
```

## Comando de verificacao pos-setup

```bash
# Verificar que tudo esta no lugar
echo "=== Prisma version ===" && npx prisma --version
echo "=== DATABASE_URL ===" && grep DATABASE_URL .env
echo "=== Schema file ===" && cat prisma/schema.prisma
```
