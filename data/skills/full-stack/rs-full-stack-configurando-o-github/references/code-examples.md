# Code Examples: Configurando o Perfil do GitHub

## Exemplos de Bio

### Iniciante
```
Estudante de desenvolvimento Full Stack | Aprendendo todos os dias
```

### Junior
```
Desenvolvedor Front-end | React & TypeScript | Apaixonado por UI/UX
```

### Pleno
```
Full Stack Developer @empresa | Node.js, React, PostgreSQL
```

### Senior
```
Staff Engineer @bigtech | Open source contributor | Building developer tools
```

## GitHub Profile README (bonus)

Alem das configuracoes basicas da aula, voce pode criar um repositorio com o mesmo nome do seu usuario para ter um README no perfil:

```markdown
# Ola! 👋

Sou desenvolvedor Full Stack focado em React e Node.js.

- 🔭 Trabalhando atualmente na [Empresa]
- 🌱 Aprendendo TypeScript e testes automatizados
- 📫 Como me encontrar: [LinkedIn](https://linkedin.com/in/seu-perfil)
```

Isso nao foi abordado na aula, mas e uma extensao natural da configuracao de perfil.

## Checklist rapido de verificacao

```
[ ] Nome preenchido
[ ] Bio preenchida
[ ] Localizacao (minimo: Brasil)
[ ] Pelo menos 1 rede social (preferencialmente LinkedIn)
[ ] Foto de perfil (< 1MB)
[ ] Salvar alteracoes
[ ] Verificar perfil publico
```
